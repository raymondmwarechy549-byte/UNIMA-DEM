📋 Required Email Format

Voters must register with a valid UNIMA institutional email matching the pattern below.

Accepted Format
dept/spec/YY/NNN@unima.ac.mw bsoc/pps/03/23@unima.ac.mw bsoc/dem/26/001@unima.ac.mw
Pattern: [letters]/[letters]/[2 digits]/[2–3 digits]@unima.ac.mw
dept = department code  ·  spec = specialism  ·  YY = intake year  ·  NNN = student number
